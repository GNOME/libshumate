libshumate
==========

libshumate is a GTK4 widget to display maps.

libshumate requires:

   * glib >= 2.68
   * gio >= 2.68
   * gobject >= 2.68
   * gobject-introspection >= 0.6.3
   * cairo >= 1.4
   * sqlite3 >= 1.12.0
   * libsoup >= 2.42 OR libsoup >= 3.0 if `-Dlibsoup3=true` is passed to meson
   * gtk4

Building is done with [Meson](https://mesonbuild.com).
